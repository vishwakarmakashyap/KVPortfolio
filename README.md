# Kashyap Vishwakarma - Portfolio Website

A modern, responsive portfolio website showcasing professional experience, skills, and projects.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Dark theme with gradient accents and smooth animations
- **Interactive Elements**: Hover effects, smooth scrolling, and animated sections
- **Professional Layout**: Clean sections for experience, skills, projects, and contact

## Technologies Used

- HTML5
- CSS3 (with CSS Variables and Animations)
- JavaScript (Vanilla)
- Font Awesome Icons

## How to Use

1. Open `index.html` in your web browser
2. Navigate through sections using the navigation menu
3. All sections are responsive and mobile-friendly

## Sections

- **Home**: Hero section with introduction
- **About**: Professional summary and contact information
- **Experience**: Work history with detailed responsibilities
- **Skills**: Technical skills organized by category
- **Projects**: Portfolio of completed projects
- **Education**: Academic background and achievements
- **Contact**: Contact information and links

## Customization

To customize the website:

1. **Colors**: Edit CSS variables in `styles.css` (`:root` section)
2. **Content**: Update text in `index.html`
3. **Social Links**: Add your LinkedIn/GitHub URLs in the social links section

## Browser Support

- Chrome (recommended)
- Firefox
- Safari
- Edge
